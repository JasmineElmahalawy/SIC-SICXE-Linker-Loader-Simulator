# 📦 SIC / SIC-XE Loader Simulator  
### System Programming Course Project – CC410  
**Arab Academy for Science, Technology & Maritime Transport (AAST)**

This project implements a **Loader Simulator for SIC and SIC/XE object programs**, developed as part of the **CC410 System Programming course** at the **Arab Academy for Science, Technology and Maritime Transport (AAST)**.

The application provides a **graphical interface for loading and interpreting HTE (Header, Text, End) object files**, simulating the loading process used in classical system programming environments.

---

# Features

- Support for **SIC and SIC/XE architectures**
- Load and parse **HTE object program files**
- Simulate the **loader process**
- Interactive **GUI for file selection and execution**
- Displays processed output after loading
- Built using **Python**

---

# 🖥️ Application Workflow

The program follows these steps:

1. Run the Python application
2. Choose the architecture (**SIC or SIC/XE**)
3. Upload the **HTE object file**
4. The loader processes the file
5. The result is displayed in the GUI
